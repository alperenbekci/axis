# Axis Agent Landing Page

![MultiversX AI Chat](public/logo.png)

This is the official landing page for Axis Agent, built with Next.js, ShadCN, and Tailwind CSS. It provides an overview of Axis Agent's capabilities, features, and use cases, designed with a modern and responsive UI.

## 🛠️ Tech Stack

- **Next.js** – For server-side rendering and optimized performance
- **ShadCN** – For reusable and accessible UI components
- **Tailwind CSS** – For clean and responsive design

## ✨ Features

- **Fully Responsive UI** – Optimized for all devices
- **Fast Performance** – Built with Next.js for seamless navigation
- **Dark Mode Support** – Automatic theme switching based on user preference
- **SEO Optimized** – Structured for better visibility on search engines

## ⚙️ Installation

1. Clone the repository:
```bash
git clone https://github.com/your-repo/axisagent-landing.git
cd axisagent-landing
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

The site will be available at http://localhost:3000.

## 🚀 Deployment

For production, build and start the project:

```bash
npm run build
npm start
```

## 🎨 Customization

- Modify styles in `/styles/globals.css`
- Update components inside `/components`
- Edit content in `/pages/index.tsx` for homepage customization
